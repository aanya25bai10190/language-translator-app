# 🌍 Language Translator App

A simple and interactive **multilingual translation web application** built using **Streamlit** and **Deep Translator API**. This app allows users to translate text between multiple languages with a clean and user-friendly interface.

---

## 🚀 Features

* 🌐 Translate text between multiple languages
* 🔄 Swap source and target languages
* 🎤 Voice input support (via system voice typing)
* 📋 Copy/download translated text
* 🔢 Character count display
* 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

* **Frontend & Backend:** Streamlit
* **Language:** Python
* **Translation API:** Deep Translator

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/aanya25bai10190/language-translator-app
cd language-translator-app
```

2. Create virtual environment (optional but recommended):

```bash
python -m venv venv
venv\Scripts\activate
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the app:

```bash
streamlit run app.py
```
